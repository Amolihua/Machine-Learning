# 🤖 Prácticas de Machine Learning

Bienvenido a mi laboratorio de modelos predictivos. 

En lugar de solo leer la teoría estadística detrás de los modelos, aquí los pongo a prueba usando código en Python. Este repositorio es mi bitácora personal donde entreno, evalúo y comparo diferentes algoritmos paso a paso.

## 🗂️ ¿Cómo está organizado este laboratorio?

He dividido mis prácticas en las dos grandes ramas del aprendizaje automático:

* **`/Supervisado`**: Aquí guardo los modelos donde le enseñamos a la máquina con ejemplos claros (datos etiquetados). Actualmente puedes encontrar mis prácticas con **Árboles de Decisión**, explorando cómo el algoritmo toma caminos lógicos para predecir un resultado.
* **`/No Supervisado`**: La sección donde dejo que los algoritmos encuentren patrones ocultos por sí mismos. Ahora mismo estoy armando un cuaderno sobre **Clustering (Agrupamiento)** que sigo desarrollando y puliendo.
* **`/Plantillas`**: Mi bóveda de código reutilizable. Como buen programador, no me gusta escribir lo mismo dos veces. Aquí guardo estructuras base (como mi plantilla actualizada para Árboles de Decisión) que me permiten importar datos, entrenar y evaluar modelos mucho más rápido en futuros proyectos.

## 🛠️ Mi Stack de Machine Learning

* **Lenguaje:** Python
* **Librerías principales:** Scikit-Learn (para los modelos), Pandas y NumPy (para preparar la data antes del entrenamiento).
* **Enfoque:** Entender la matemática y estadística detrás de las herramientas para no usarlas como simples "cajas negras".


Siéntete libre de revisar mi código. Mi objetivo es que este repositorio siga creciendo a medida que domino modelos más complejos.
